# Euro Foot Checker
<p>Euro Foot Checkerは、欧州サッカーの異なるリーグにある好きなチームの試合結果を一目で確認したい、欧州サッカーファン向けのサッカーの試合結果配信サービスです。</p>

<p>ユーザーは異なるリーグにある好きなチームを登録し、試合結果を一つのページで確認することが出来ます。</p>

## サービスURL
https://euro-foot-checker.herokuapp.com

## 開発環境
- Ruby 3.0.1
- Rails 6.1.4

## 利用方法

### インストール
```
$ bin/setup
$ rails server
```

### 初期データ
```
 $ rails db:seed
```

### テスト
```
$ bundle exec rspec
```

### lint
```
$ rubocop
```
